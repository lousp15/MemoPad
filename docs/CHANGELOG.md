# MemoPad 变更日志

## [v0.1] — 初始版本

### 新增
- 基础备忘录 CRUD（创建/编辑/删除/筛选）
- 本地持久化存储（IndexedDB + Dexie.js）
- 会话级无限撤销/重做（Ctrl+Z / Ctrl+Y）
- 结束会话二次确认
- 关闭弹窗三选一（保存并退出 / 不保存退出 / 取消）
- MUI 6.x 主题自定义（亮/暗色 + CSS 变量调色）
- GitHub 云端同步（安全模式 + 强制覆盖模式）
- 冲突解决 UI（逐条选择覆盖方向）
- 写入失败重试（指数退避 3 次 + 紧急备份）
- 进程崩溃后数据恢复（EmergencyStorage）

### 已知限制
- **备忘录列表无虚拟化**：当前使用普通 DOM 列表渲染。
  最大上限 500 条时约产生 500 个 DOM 节点，现代浏览器可流畅渲染。默认上限 15 条无性能风险。
  **后续优化**：若用户反馈大批量数据渲染卡顿，引入 `react-window` 实现虚拟化列表。
- **提醒机制无系统级通知**：仅应用启动时检查并标记过期条目。
  关闭应用后无通知，需保持应用前台运行。
- **GitHub 同步依赖用户手动配置**：需自行创建 Personal Access Token（`repo` 权限）。
